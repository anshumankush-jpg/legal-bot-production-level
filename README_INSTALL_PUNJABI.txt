                    INSTALL PUNJABI VOICE - QUICK START
================================================================================

EASIEST WAY TO INSTALL:
-----------------------

1. Double-click: INSTALL_PUNJABI.bat
2. Click "Yes" when asked for Administrator permission
3. Follow the prompts
4. Restart your computer when done

That's it!

================================================================================

ALTERNATIVE METHODS:
--------------------

Method 1: PowerShell Script
   - Right-click PowerShell -> "Run as Administrator"
   - cd to this folder
   - Run: .\install_punjabi_voice.ps1

Method 2: Windows Settings
   - Win + I -> Time & Language -> Language
   - Add a language -> Search "Punjabi"
   - Install Punjabi (India)
   - Download Speech pack
   - Restart computer

================================================================================

AFTER INSTALLATION:
-------------------

1. RESTART YOUR COMPUTER (Important!)

2. Test the voice:
   - Open: tests\check_browser_voices.html
   - Look for "Google ਪੰਜਾਬੀ"
   - Click "Test" button

3. Test in demo:
   - Open: tests\voice_settings_demo.html
   - Select Punjabi
   - Test voice

================================================================================

TROUBLESHOOTING:
----------------

Problem: Voice not available after install
Solution: 
   1. Restart computer (not just browser!)
   2. Restart browser
   3. Clear browser cache

Problem: Installation fails
Solution:
   1. Run PowerShell as Administrator
   2. Check internet connection
   3. Try manual Windows Settings method

Problem: Still not working
Solution:
   1. Run: python tests\diagnose_punjabi_voice.py
   2. Read: INSTALL_PUNJABI_GUIDE.md
   3. Use OpenAI TTS as fallback
