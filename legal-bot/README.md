## ğŸ”§ Installation

### Prerequisites

- **Node.js** 14+ and npm
- **Python** 3.8+
- **Tesseract OCR** (optional, for image processing)
- **OpenAI API Key** (for AI features)

### Step 1: Clone the Repository

```bash
git clone https://github.com/anshumankush-jpg/legal-bot-production-level.git
cd legal-bot-production-level
```

### Step 2: Install Backend Dependencies

```bash
cd backend
pip install -r requirements.txt
```

### Step 3: Install Frontend Dependencies

```bash
cd frontend
npm install
npm install jspdf  # For PDF generation
```

### Step 4: Configure Environment

Create a `.env` file in the `backend` directory:

```bash
# Required
OPENAI_API_KEY=your_openai_api_key_here

# Optional - External Legal APIs
LEGALZOOM_API_KEY=your_legalzoom_key
LEXISNEXIS_API_KEY=your_lexisnexis_key
CASETEXT_API_KEY=your_casetext_key

# Optional - Security
JWT_SECRET_KEY=your_secret_key_here

# Optional - Settings
LLM_PROVIDER=openai
OPENAI_CHAT_MODEL=gpt-4o-mini
LOG_LEVEL=INFO
```

### Step 5: Start the Application

**Terminal 1 - Backend:**
```bash
cd backend
python -m uvicorn app.main:app --reload --port 8000
```

**Terminal 2 - Frontend:**
```bash
cd frontend
npm run dev
```

### Step 6: Access the Application

Open your browser and navigate to:
- **Frontend:** http://localhost:4200 or http://localhost:5173
- **Backend API:** http://localhost:8000
- **API Docs:** http://localhost:8000/docs
