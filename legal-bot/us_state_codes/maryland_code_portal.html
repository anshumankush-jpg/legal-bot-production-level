    You are about to download a "comma-separated values" (CSV) file and/or a JSON file.
    The file includes information such as number, sponsor, title, legislative status, synopsis, committee assignments, legislative history, hearing dates, etc for each piece of legislation introduced during the selected legislative session.
    A word about the file format: CSV and JSON files are common file formats, widely supported by consumer and business applications and is used to move data between programs. The file you download can be imported into Excel, a text editor, browser, or any other application that supports the CSV and JSON formats. To save, right-click button and save as desired file format.
